A practice project for publishing Python projects created with uv to PyPI with PEP740 signatures. Contains only a hello() function that returns "Hello!".

## Installation

```console
pip install h4-hello
```

## Usage

```console
$ h4-hello
Hello!
```

or

```python
from h4_hello.hello import hello
print(hello())
```
